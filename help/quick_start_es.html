InSightec Service Hub - Inicio rápido

  1. Abra la pantalla Herramientas.
  2. Instale el ZIP de cada herramienta con Instalar ZIP, si es necesario.
  3. Confirme el estado con Validar herramientas.
  4. Use Iniciar para abrir la herramienta seleccionada.
  5. Si encuentra un problema, abra Comentarios y adjunte logs o capturas de pantalla.

Las actualizaciones se ejecutan con permisos de usuario normal.
