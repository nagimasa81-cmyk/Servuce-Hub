InSightec Service Hub v1.1.1-M1-FEU

Goal:
- Field evaluation build focused on Tool Launcher and Feedback.

Run:
1. Extract this ZIP.
2. Run Run_Hub.bat.
3. Open Tools.
4. For each tool, click Install ZIP and select the corresponding tool ZIP.
5. Click Validate Tools.
6. Launch each tool from the card.
7. Use Feedback to create Outlook draft or email template.

Notes:
- No administrator permission is required.
- If the EXE is inside a nested folder after ZIP extraction, the Hub searches recursively.
- Validation reports are saved under logs/.
- Feedback templates and screenshots are saved under feedback/.
